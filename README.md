# vue-multiple-mock-env

一个使用vue的多环境webpack模版，基于vue-cli@4，引用mockjs模拟http请求和响应，可在控制台network返回响应数据

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
